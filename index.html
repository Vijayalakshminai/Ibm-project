Hosting a Website with Docker & Nginx

What is Docker?

Docker is a containerization platform that allows developers to package applications along with their dependencies in isolated environments. This ensures that applications run consistently across different computing environments.

How Docker Works

Docker uses containers, which are lightweight, stand-alone executable packages that include everything needed to run a piece of software, including the code, runtime, libraries, and dependencies. Containers are built from images using Dockerfiles, which define the application's setup and configuration.

Steps to Host a Website with Nginx in Docker

1. Create a simple HTML website and place it in a directory.
2. Write a Dockerfile specifying the Nginx base image and copying the website files into the container.
3. Build a Docker image from the Dockerfile using the command: docker build -t my-nginx-site .
4. Run the container using: docker run -d -p 8080:80 my-nginx-site
5. Access the website via http://localhost:8080 in a web browser.
